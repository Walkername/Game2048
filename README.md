# Game2048
3 packages: 
1. graphics_pack; 
2. controller_pack; 
3. logic_pack.

Third task (coursework) for credit.

Also implemented tests for logic_pack: (GameBoard, Logic classes).

One fxml file is used "main.fxml".


Control:

R -> start or restart the game;

W -> shift cells UP;

S -> shift cells DOWN;

A -> shift cells LEFT;

D -> shift cells RIGHT;
